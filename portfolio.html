  • Connect 4 Game

    This game is an in-browser and online version of classic Connect 4 game made with JavaScript, jQuery, HTML5, and CSS3 technologies. To start the game, the players need to drop the token to the very bottom row and start the game from the bottom row. Otherwise the token won't be dropped. Players can take truns after each droped one token. The player who could connect four tokens either vertically or horizontally, will be the winner!

  • Trelp

    Trelp is a web application that allows users to keep track of their trips to around the world, and share their experience and memories of each location they have visited with other users. Trelp is a group project and is created with Angular JS, Express, HTML, and CSS technologies. To start, the user needs to sign up and make a new user account. After authentication, the user will be able to add, edit, and delete trips into/from their account.

  • Don't Forget To:

    This is a very basic and simple Todo list application made with Mango, Express, Angular, and Node technologies without authenticatio and authorization. User can add a new Todo to the list by clicking on the Add Todo button. Each Todo can also be edited and deleted easily if needed.
